MV Lyrics Animation Maker

GitHub Pagesでそのまま公開できる、歌詞アニメーション作成支援用ブラウザアプリです。

ファイル構成
- index.html
- style.css
- app.js
- README.txt

主な機能
- 音声ファイル / MP4読み込み
  - MP4はプレビューキャンバス内で歌詞の下敷き動画として表示
- 歌詞TXT読み込み
  - TXTの1行を1つの歌詞ボタンとして追加
  - 空行は自動除外
- 音声 / MP4再生位置に合わせた歌詞入力
  - 音声またはMP4再生中、任意のタイミングで歌詞ボタンをクリックすると、その時刻にフレーズを配置
  - 入力済みボタンはグレーアウト
  - 入力済みボタンをもう一度クリックすると、そのフレーズの配置をキャンセル
- フレーズごとの個別設定
  - 開始秒 / 終了秒
  - 位置X / 位置Y
  - 左寄せ / 中央寄せ / 右寄せ
  - フォント / フォントサイズ / 文字色 / 行間
  - 通常表示
  - タイプライター表示
  - ジャンプしながらタイプライター表示
  - タイプライター速度
  - ジャンプの大きさ
  - ジャンプの速さ
  - フェードイン / フェードアウト
  - 縁取りの色 / 太さ
  - ドロップシャドウの色 / ぼかし / X / Y
- 基本設定
  - 基本フォント、基本サイズ、初期表示秒数、初期アニメーション、初期文字揃え
  - 最初のフレーズには基本設定を反映
  - 2つ目以降のフレーズは、選択中または直前に入力したフレーズの設定を引き継ぎ
- プレビュー背景
  - 背景色
  - 任意の背景画像
  - 画面いっぱい / 全体表示 / 引き伸ばし
  - 背景画像サイズのスライダー調整
  - 背景はプレビュー確認用で、透過PNG書き出しには含まれません
- 透過PNG連番ZIP書き出し
  - FPS、開始秒、終了秒、ファイル名接頭辞を指定可能
- JSON保存 / 読み込み
- IndexedDBキャッシュ保存 / 復元

公開方法
1. このフォルダ内のファイルをGitHubリポジトリへアップロードします。
2. GitHub Pagesを有効化します。
3. index.html がトップページとして表示されます。

注意
- 透過PNGのZIP書き出しには JSZip CDN を使用しています。
- 長尺・高FPSで書き出すと生成枚数が多くなり、ブラウザのメモリを多く使います。
- 書き出し時、背景色・背景画像・下敷きMP4は含まれません。MV編集ソフト側で背景映像や画像と合成してください。


【プリセット機能】
選択中フレーズ設定の「フレーズ設定プリセット」で、現在選択中のフレーズの見た目・位置・アニメーション設定を名前付きで登録できます。
プリセットには歌詞本文、開始秒、終了秒は含まれません。
登録したプリセットはブラウザのローカルストレージにも保存されるため、別プロジェクトでも再利用できます。JSON保存時にもプリセット情報が含まれます。

更新：追加アニメーション
- ジャンプしながら表示：全文を表示したまま、文字ごとにジャンプします。
- 登場時と退場時だけジャンプ：開始時と終了直前だけ文字がジャンプします。ジャンプの大きさ・速さは既存スライダーを使用します。


Update: 한국어 UI / Korean lyrics / SRT UTF-8 BOM support added.


更新：動画変換アプリDL
- 書き出し欄の一番下に「動画変換アプリDL」ボタンを設置しました。
- 透過PNG連番ZIPを透過動画化（MOV）するためのWindows用変換アプリをダウンロードできます。
- WebM/AVIは環境によって透過や読み込みに問題が出るため、MOV変換を推奨します。


更新：縦書き対応
- 選択中フレーズ設定に「文字方向」を追加しました。
- 横書き / 縦書きをフレーズごとに切り替えできます。
- 縦書き時は「折り返し幅」を「折り返し高さ」として扱います。
- 縦書き列間のスライダーを追加しました。
- 英数字を横倒しにするチェックボックスを追加しました。
- 縦書きは横書き描画とは別ルートで描画し、既存の横書きアニメーションへの影響を抑えています。


[Layer update]

- Layer names and label colors can be edited.
- Placed phrase list now shows L1 / L2 style colored labels.

更新：MP4下敷きプレビュー対応
- 音声ファイル読み込みと同じ場所からMP4 / M4V / MOVを読み込めます。
- MP4読み込み時はプレビュー枠内に動画を表示し、その上に歌詞キャンバスを重ねます。
- 歌詞ボタンのタイミング入力、現在時刻プレビュー、終了秒反映はMP4の再生時間にも対応しています。
- 透過PNG連番書き出しにはMP4映像は含めず、歌詞レイヤーのみを書き出します。
